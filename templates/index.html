About

This Predictor is based on a Classification Model whose dataset is centered on cleveland.csv containing about 303 patients data including both the kinds, cardiac and non-cardiac patients. This model is created using Support Vector Machine Classifier which upon evalulating gave 90% accuracy on test set.
